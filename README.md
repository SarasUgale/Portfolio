# Saras Portfolio

Dual-pane Next.js 14 portfolio concept built from the resume in `C:/Users/SARAS/Downloads/SARAS_UGALE_RESUME(AIML).pdf`.

## Stack

- Next.js 14
- Tailwind CSS
- Framer Motion
- GSAP
- D3

## Run

1. Install dependencies with your local Node package manager:
   `npm install`
2. Start the app:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Notes

- The neural tether system is implemented with SVG plus GSAP spring-like interpolation.
- The logic-side content is mapped directly from the resume projects and skills.
- The life-side visuals use stylized placeholder panels because no workspace or gaming images were provided in the source materials.
